# jsonpath模块

```
使用场景 :多层嵌套的字典复杂数据的直接提取
使用方法:
	from jsonpath import jsonpath 
	ret = jsonpath(a,"jsonpath语法规则字符串")
语法:
	$   根节点(最外层大括号)
	.	子节点
	..	内部任意位置, 子孙节点
	
注意:jsonpath的结果为列表 ,获取数据需要索引
```

# xpath helper(谷歌插件)

```
作用:
	对当前页面测试xpath语法规则
语法:
	节点选择语法:
		html	
        /	绝对路径节点
        //	相对路径节点
        .	子节点
        ..	上一级节点
        /text()	从开闭标签里取文本内容
        /@(属性名)	从选中的节点标签抽取指定属性的值 
        /[]	节点修饰语法:
        		1,通过索引
        			[last()]	选择最后一个
        			[position]	区间选择
        		2,通过属性值
        			出现在[]中的@是使用标签属性名和属性值修饰节点,
        			出现在结尾的/@是取属性值
        		3,通过子节点的值修饰节点
        		4,通过包含修饰
        			contains(@属性值)
        		找翻译url的时候精良不要使用索引
        复合使用语法
```
